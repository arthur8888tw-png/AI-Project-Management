# DevTools 溝通訓練系統

## 專案簡介

這是一個互動式訓練系統,幫助使用者學會用正確的方式與 AI 對話,特別是在 UI/CSS 相關問題上。

## 功能特色

- ✅ 50 題精選練習題 (擴充版!)
- ✅ 5 大類別: 元素定位、數值讀取、顏色吸取、CSS 屬性、溝通模板
- ✅ 雙題型: 輸入題 + 選擇題
- ✅ 即時反饋與詳細解析
- ✅ 成績追蹤與弱點分析
- ✅ LocalStorage 儲存進度
- ✅ 響應式設計,支援手機

## 題型分布

### 輸入題 (30 題)
需要手動輸入答案,訓練精確表達能力

### 選擇題 (20 題)
觀念題,測試對 DevTools 和 CSS 的理解

## 如何使用

1. 開啟 `index.html`
2. 點擊「開始練習」
3. 根據示範網頁回答問題
4. 查看成績與弱點分析
5. 針對弱點加強練習

## 檔案結構

```
devtools-training/
├── index.html      # 主頁面
├── questions.js    # 題庫資料
├── app.js          # 應用邏輯
└── README.md       # 說明文件
```

## 題型說明

### 1. 元素定位 (2 題)
學習如何找出元素的 class、id 和 CSS selector

### 2. 數值讀取 (2 題)
練習從 DevTools 讀取 padding、margin、width 等數值

### 3. 顏色吸取 (2 題)
掌握顏色選擇器,讀取 HEX、RGB 值

### 4. CSS 屬性 (2 題)
識別 display、position、z-index 等屬性

### 5. 溝通模板 (2 題)
學會將模糊問題轉換為精確的溝通格式

## 評分標準

- 🥇 **85% 以上**: 優秀,表現出色
- 🥈 **70-84%**: 良好,繼續保持
- 🥉 **60-69%**: 及格,需要加強
- ❌ **60% 以下**: 不及格,建議重新練習

## 弱點分析

系統會自動分析你的弱點:
- 準確率 < 70% 的類別會被標記為「需要加強」
- 提供針對性的練習建議

## 技術棧

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- LocalStorage API
