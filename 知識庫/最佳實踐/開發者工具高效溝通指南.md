## 第二部分: 快速定位元素 (3 種方法)

### 方法 1: 滑鼠選取 (最快速) ⭐⭐⭐⭐⭐

**步驟**:
```
1. 按 F12 開啟 DevTools
2. 點擊左上角「選取元素」圖示 (或按 Ctrl+Shift+C)
3. 滑鼠移到想檢查的元素上
4. 點擊該元素
→ DevTools 自動定位到該元素的 HTML 和 CSS
```

**快捷鍵**:
- Windows/Linux: `Ctrl + Shift + C`
- Mac: `Cmd + Shift + C`

**優勢**:
- ⚡ 最快速 (2 秒內定位)
- 👁️ 視覺化選取
- 🎯 精確定位

**適用場景**: 90% 的情況

---

### 方法 2: 搜尋文字 (找不到元素時) ⭐⭐⭐⭐

**步驟**:
```
1. 在 DevTools 的 Elements 面板
2. 按 Ctrl+F 開啟搜尋框
3. 輸入元素的文字內容
4. Enter 跳到該元素
```

**範例**:
```
要找「福至心靈籤」標題
→ 搜尋: 福至心靈籤
→ DevTools 自動定位到 <h1>福至心靈籤</h1>
```

**適用場景**: 元素被遮擋或不可見時

---

### 方法 3: Console 查詢 (進階) ⭐⭐⭐

**步驟**:
```javascript
// 在 Console 輸入
document.querySelector('.class-name')
// 或
document.getElementById('element-id')

// 右鍵結果 → Reveal in Elements panel
```

**適用場景**: 需要批次查詢或程式化操作

## 第三部分: 獲取關鍵數值 (5 大類)

### 1. 尺寸數值

**位置**: Elements → Computed → Box Model

```
┌─────────────────────────────┐
│      Margin (外邊距)         │
│  ┌─────────────────────┐   │
│  │   Border (邊框)      │   │
│  │  ┌───────────────┐  │   │
│  │  │ Padding (內距) │  │   │
│  │  │  ┌─────────┐  │  │   │
│  │  │  │ Content │  │  │   │
│  │  │  │ 200×100 │  │  │   │
│  │  │  └─────────┘  │  │   │
│  │  │   10 10 10 10 │  │   │
│  │  └───────────────┘  │   │
│  │    2  2  2  2       │   │
│  └─────────────────────┘   │
│      20 20 20 20           │
└─────────────────────────────┘
```

**如何讀取**:
```
Content: 200 × 100 (寬 × 高)
Padding: 10 (上) 10 (右) 10 (下) 10 (左)
Border: 2 (四邊)
Margin: 20 (四邊)

總寬度 = 200 + 10×2 + 2×2 + 20×2 = 264px
總高度 = 100 + 10×2 + 2×2 + 20×2 = 164px
```

**與 AI 溝通範例**:
```
❌ "這個元素太小了"
✅ "header 元素當前寬度 200px,需要增加到 300px"

❌ "間距太大"
✅ "header 的 padding-top 當前 20px,需要減少到 10px"
```

---

### 2. 顏色數值

**位置**: Elements → Styles → 點擊顏色方塊

**顏色格式**:
```css
/* HEX (最常用) */
color: #ff0000;

/* RGB */
color: rgb(255, 0, 0);

/* RGBA (帶透明度) */
color: rgba(255, 0, 0, 0.5);

/* HSL (色相、飽和度、亮度) */
color: hsl(0, 100%, 50%);
```

**DevTools 顏色選擇器功能**:
```
1. 點擊顏色方塊 → 開啟選擇器
2. 吸管工具 → 從頁面任何地方取色
3. 對比度檢查 → 確保可讀性
4. 格式切換 → HEX ↔ RGB ↔ HSL
```

**與 AI 溝通範例**:
```
❌ "這個顏色太淡了"
✅ "header 背景色當前 #f0f0f0,需要改為 #450a0a"

❌ "配色不協調"
✅ "box-shadow 顏色從 rgba(0,0,0,0.5) 改為 rgba(69,10,10,0.3)"
```

---

### 3. 位置數值

**位置**: Elements → Computed → Position

**關鍵屬性**:
```css
position: relative | absolute | fixed | sticky
top: 20px
left: 10px
transform: translateY(-50%)
z-index: 9999
```

**與 AI 溝通範例**:
```
❌ "這個元素位置不對"
✅ "圓牌當前 transform: translateY(-40%),需要改為 translateY(-12%)"

❌ "被遮住了"
✅ "動畫層級 z-index: 100,需要提升到 9999"
```

---

### 4. 字體數值

**位置**: Elements → Computed → Font

**關鍵屬性**:
```css
font-family: "Noto Serif TC", serif
font-size: 24px
font-weight: 700
line-height: 1.5
letter-spacing: 0.05em
```

**與 AI 溝通範例**:
```
❌ "字太小了"
✅ "標題 font-size 當前 24px,需要增加到 32px"

❌ "字體不對"
✅ "font-family 從 Arial 改為 'Noto Serif TC'"
```

---

### 5. 動畫數值

**位置**: Elements → Styles → animation 屬性

**關鍵屬性**:
```css
animation-name: scroll-unroll
animation-duration: 2.2s
animation-timing-function: cubic-bezier(0.1, 0, 0.3, 1)
animation-delay: 0s
animation-iteration-count: 1
animation-direction: normal
animation-fill-mode: forwards
```

**與 AI 溝通範例**:
```
❌ "動畫太快了"
✅ "scroll-unroll 動畫當前 duration: 1.5s,需要延長到 2.2s"

❌ "動畫不自然"
✅ "timing-function 從 ease 改為 cubic-bezier(0.1, 0, 0.3, 1)"
```

## 第八部分: 常見問題 FAQ

### Q1: 如何找到元素的 CSS selector?

**A**: 
```
1. 在 Elements 面板選中元素
2. 右鍵 → Copy → Copy selector
3. 得到類似: #header > div.container > h1
```

### Q2: 如何知道哪個 CSS 屬性在起作用?

**A**:
```
Styles 面板中:
- 沒有刪除線 = 正在生效
- 有刪除線 = 被覆蓋
- 灰色 = 繼承自父元素
```

### Q3: 如何測試修改效果?

**A**:
```
1. 在 Styles 面板直接編輯 CSS
2. 即時看到效果
3. 滿意後再告訴 AI 修改
```

### Q4: 如何匯出所有 CSS?

**A**:
```
1. 選中元素
2. Styles 面板右鍵 → Copy all declarations
3. 或使用 Computed 面板 → Show all
```

### Q5: 如何查看動畫細節?

**A**:
```
1. More tools → Animations
2. 播放/暫停動畫
3. 調整速度
4. 查看 timing function 曲線
```

## 第九部分: 進階技巧

### 技巧 1: 使用 Console 批次查詢

```javascript
// 查詢所有特定 class 的元素
document.querySelectorAll('.button').forEach(el => {
  console.log({
    text: el.textContent,
    width: el.offsetWidth,
    height: el.offsetHeight,
    color: getComputedStyle(el).color
  });
});
```

### 技巧 2: 監聽樣式變化

```javascript
// 監聽元素樣式變化
const observer = new MutationObserver((mutations) => {
  mutations.forEach(mutation => {
    if (mutation.attributeName === 'style') {
      console.log('樣式已變更:', mutation.target.style.cssText);
    }
  });
});

observer.observe(element, {
  attributes: true,
  attributeFilter: ['style']
});
```

### 技巧 3: 匯出完整樣式表

```javascript
// 匯出所有 CSS 規則
Array.from(document.styleSheets).forEach(sheet => {
  try {
    Array.from(sheet.cssRules).forEach(rule => {
      console.log(rule.cssText);
    });
  } catch(e) {
    console.log('無法存取:', sheet.href);
  }
});
```
