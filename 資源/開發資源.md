## 📖 最佳實踐

### 命名規範
- **檔案**: kebab-case (例: user-profile.tsx)
- **元件**: PascalCase (例: UserProfile)
- **函數**: camelCase (例: getUserData)
- **常數**: UPPER_SNAKE_CASE (例: API_KEY)

### Git Commit 訊息
\\\
feat: 新增功能
fix: 修復 bug
docs: 文檔更新
style: 程式碼格式調整
refactor: 重構
test: 測試
chore: 雜項
\\\
